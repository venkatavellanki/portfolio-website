# Venkat Vellanki | Portfolio Web Application

## Overview
Hi! I’m **Venkat Vellanki**, an **Assistant System Engineer (ASE)** and **Python Full Stack Developer**.  
This is my professional portfolio web application, where I showcase my **education, skills, work experience, projects, certifications, and contact information** in **15 interactive sections**.

I designed this web application to be:
- Modern and visually appealing  
- Responsive on desktop and mobile  
- Interactive, with smooth scrolling and active section highlighting  
- Easy to navigate and professional-looking for certification purposes
